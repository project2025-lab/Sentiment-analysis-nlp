# Sentiment Analysis using NLP

This project performs sentiment classification on text data using Natural Language Processing (NLP) techniques. The model classifies text into Positive, Negative, and Neutral categories.

---

## 📌 Project Overview

The goal of this project is to build a machine learning model that can automatically determine the sentiment of textual data.

The workflow includes:
- Data cleaning and preprocessing
- Text vectorization using TF-IDF
- Model training and evaluation
- Performance analysis

---

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- scikit-learn
- Pandas
- NumPy
- NLTK / Text preprocessing
- Matplotlib / Seaborn (for visualization)
